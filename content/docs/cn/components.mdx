## LinkButton

链接按钮组件，用于链接 URL 并显示文本。

```mdx
<LinkButton url="https://github.com/showlotus/fumadocs-template" text="Github" />
```

## ThemeImage

图片组件，用于在亮色和暗色模式下显示不同的图片。

```mdx
<ThemeImage light="../images/sun.svg" dark="../images/moon.svg" className="w-10" />
```

点击 <ToggleTheme /> 切换主题，图标会随之变化。
